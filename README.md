# 席替えアプリ (Seating Arrangement App)

学校の席替えを支援する Web アプリです。座席数や優先席を設定し、生徒の座席をランダムに抽選できます。

## 技術スタック

- Next.js 16（App Router / `output: "export"` による静的エクスポート）
- React 19
- TypeScript
- Tailwind CSS

## セットアップ

```bash
npm install
```

## 開発サーバー

```bash
npm run dev
# http://localhost:3000
```

## 静的エクスポート（ビルド）

```bash
npm run build
```

`out/` ディレクトリに静的ファイルが生成されます。任意の静的ホスティング（GitHub Pages、S3、Netlify など）にそのまま配置できます。

## 機能

### 生徒一覧（サイドメニュー: 生徒一覧タブ）

- **テンプレート出力**: 40 名分（生徒1〜生徒40、全員「男」、予約座席なし）の CSV を出力します。
- **インポート**: CSV を読み込みます。不整合があれば原因を一覧表示します。
- **エクスポート**: 現在の生徒一覧を CSV 出力します。
- 各生徒の行には出席番号・名前・座席番号（未決定はハイフン）を表示します。
- **抽選 / 指定 / 取消** ボタンで座席を操作します。

### 座席設定（サイドメニュー: 座席設定タブ）

- 縦列・横行を 1〜8 で設定（デフォルト 縦6列・横6行）。
- 座席の種類をラジオボタンで選び、メインの座席をクリックして設定します。
  - 指定なし / 男子席 / 女子席 / 使用不可（互いに排他）
  - 優先席（男子席・女子席・指定なしと重複可。再クリックで解除）

### 座席表示（メイン）

- 上部に「黒板」を表示。
- 各座席に `No.◯`（連番）と生徒名を表示。
- 種類ごとの配色: 指定なし=ベージュ / 男子席=青 / 女子席=赤 / 使用不可=グレー（番号・名前は非表示で×印）。優先席は左上に緑の星。
- 着席中の座席は右上の × で取消できます。
- 「抽選」を押すと候補席がランダムに点滅した後、座席が確定します。
- 「指定」を押すと指定モードになり、メインの座席表から空席をクリックしてその生徒の座席を確定できます。

### 抽選ロジック

候補となる座席は次の優先順位で決まります（他の生徒が予約している座席は対象外）。

1. 優先席に空きがあれば優先席から抽選。ただし男子席・女子席を兼ねる優先席は性別が一致する生徒のみが対象（指定なし兼用の優先席は誰でも対象）。性別が一致する優先席が無い場合は次の手順にフォールバック。
2. 男子（女子）生徒で男子席（女子席）に空きがあればそこから抽選。
3. それ以外は空いている指定なし席から抽選。
4. 上記がすべて無い場合は、空いている座席（性別不問）から抽選。

最終的な座席は次のルールで決まります。

- 予約座席番号が設定されていて、その席が空いていれば必ずその席に決定します（点滅演出は上記の候補ルールに従うため、予約席自体は点滅候補に影響しません）。
- 予約座席が埋まっている場合は、上記の抽選ロジックにフォールバックします。
- 他の生徒が予約している座席は、抽選・指定のいずれでも候補になりません（予約者のために確保されます）。

## CSV フォーマット

```
出席番号,名前,性別,予約座席番号
1,生徒1,男,
2,生徒2,女,5
```

- 出席番号: 重複のない半角数字
- 性別: 「男」または「女」
- 予約座席番号: 重複のない半角数字、または未入力

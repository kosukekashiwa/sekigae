# 席替えアプリ (Seating Arrangement App)

学校の席替えを支援する Web アプリです。座席数や優先席を設定し、生徒の座席をランダムに抽選できます。

## アプリの概要

画面左側のサイドメニューから「生徒一覧」と「座席設定」を切り替えて使います。

- **生徒一覧タブ**: 生徒の登録・編集や、座席の抽選・指定を行います。
- **座席設定タブ**: 座席の縦列・横行数や、座席の種類（男子席・女子席・使用不可・優先席）を設定します。

設定が終わったら、メイン画面の座席表から各生徒の座席を「抽選」または「指定」で決めていきます。

## 生徒一覧の使い方

サイドメニューの「生徒一覧」タブで、生徒の登録と座席決めを行います。

1. 画面右上などにある「生徒一覧編集」へ進み、生徒を登録します。
   - 「テンプレート出力」を押すと、40 名分（生徒1〜生徒40、全員「男」、予約座席なし）のひな形 CSV をダウンロードできます。これを編集して使うと簡単です。
   - 編集した CSV は「インポート」から読み込みます。データに不整合があれば、原因が一覧で表示されます。
   - 表の「行を追加」「削除」からも生徒を直接編集できます。
   - 編集が終わったら「編集完了」で保存します。

   CSV のフォーマットは次の通りです。

   ```
   出席番号,名前,性別,予約座席番号
   1,生徒1,男,
   2,生徒2,女,5
   ```

   - 出席番号: 重複のない半角数字
   - 性別: 「男」または「女」
   - 予約座席番号: 特定の席に座らせたい場合に指定（重複のない半角数字、未入力可）

2. 生徒一覧の各行には、出席番号・名前・座席番号（未決定はハイフン）が表示されます。
   - **抽選**: その生徒の座席をランダムに決めます。
   - **指定**: メイン画面の座席表から空席をクリックして、その生徒の座席を直接決めます。
   - **取消**: 決まった座席を取り消し、未決定に戻します。

### 抽選ルール

- 予約座席番号が設定されている生徒は、その席が空いていれば必ずその席に決まります。埋まっている場合は通常の抽選にフォールバックします。
- 他の生徒が予約している座席は、抽選・指定のどちらでも対象外になります（予約者のために確保されます）。
- 優先席に空きがある場合はそこから優先的に抽選されます（性別が一致する優先席が優先されます）。
- 優先席が無い場合は、性別に対応する座席（男子席・女子席）、次に指定なし席、最後にそれ以外の空席の順で抽選されます。

## 座席設定の使い方

サイドメニューの「座席設定」タブで、座席表のレイアウトと座席の種類を設定します。

1. 縦列・横行の数を 1〜8 の範囲で設定します（デフォルトは縦6列・横6行）。
2. 設定したい座席の種類をラジオボタンで選び、メイン画面の座席表で対象の座席をクリックすると種類が反映されます。
   - **指定なし / 男子席 / 女子席 / 使用不可**: いずれか一つを設定（互いに排他）。
   - **優先席**: 上記の種類と重複して設定できます。再度クリックすると解除されます。

座席表では種類ごとに色分けされて表示されます。

- 指定なし = ベージュ / 男子席 = 青 / 女子席 = 赤 / 使用不可 = グレー（番号・名前は非表示で×印）
- 優先席には左上に緑の星が表示されます
